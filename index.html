⚙️ Configurar Sincronización

📋 Paso a paso:
1. Abrí script.google.com con la cuenta compartida
2. Creá un Nuevo Proyecto y pegá el código del archivo Code.gs
3. Hacé clic en Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
5. Copiá la URL que te da e ingresala abajo

⚠️ Para usar sin Netlify: abrí una terminal en la carpeta del HTML y ejecutá:
python -m http.server 8080
Luego entrá a http://localhost:8080/cuenta_corriente_app.html
